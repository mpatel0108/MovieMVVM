# MovieMVVM

An android movie app that fetches movie list from a REST API.

Built with ->

* Kotlin
* MVVM architectural pattern
* RecyclerView
* CardView
* Glide
* Paging library for Pagination
* RxJava
* Retrofit
* Internet Connection
* themoviedb.org REST API!
